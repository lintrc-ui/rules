Visual configuration of code linters

This resource was created to simplify customization of popular code linters among developers. To facilitate and speed up the process, all settings are divided into logical sections and have a visual display of the applicable rules.

Code linting is a type of static analysis that is frequently used to find problematic patterns or code that doesn't adhere to certain style guidelines. There are code linters for most programming languages, and compilers sometimes incorporate linting into the compilation process.

Basic information about the presented linters

  • EditorConfigPug LintHTMLHintSass Lint stylelintESLint
  • EditorConfig

    About

    EditorConfig helps maintain consistent coding styles for multiple developers working on the same project across various editors and IDEs.

    Configuration file

    For configuration, the .editorconfig file is used, which should be located in the root directory of the project.

    For more information, visit the official website of the EditorConfig project.

  • Pug Lint

    About

    An unopinionated and configurable linter and style checker for Pug (formerly Jade).

    Configuration file

    For configuration, the .pug-lintrc.json file is used, which should be located in the root directory of the project.

    For more information, visit the official website of the pug-lint project.

  • HTMLHint

    About

    HTMLHint is a Static Code Analysis Tool for HTML, you can use it with IDE or in build system..

    Configuration file

    For configuration, the .htmlhintrc file is used, which should be located in the root directory of the project.

    For more information, visit the official website of the HTMLHint project.

  • Sass Lint

    About

    A Node-only Sass linter for both sass and scss syntax!

    Configuration file

    For configuration, the .sasslintrc file is used, which should be located in the root directory of the project.

    For more information, visit the official website of the Sass Lint project.

  • stylelint

    About

    A mighty, modern linter that helps you avoid errors and enforce conventions in your styles.

    Configuration file

    For configuration, the .stylelintrc.json file is used, which should be located in the root directory of the project.

    For more information, visit the official website of the stylelint project.

  • ESLint

    About

    ESLint is an open source JavaScript linting utility.

    Configuration file

    For configuration, the .eslintrc.json file is used, which should be located in the root directory of the project.

    For more information, visit the official website of the ESLint project.
